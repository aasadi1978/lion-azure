InstallationÂ¶

  • Make sure you are granted with an access to LION_SHARED_DIR and eGIS_PE SharePoints

  • Install OneDrive app on your computer, if you have not yet done.

  • Make sure LION shared directory, namely, LION_SHARED_DIR, is available in your onedrive. To this end, visit LION_SHARED_DIR and click on Add shortcut to OneDrive to create a shortcut Documents - LION_SHARED_DIR in your OneDrive app directory. Optionally, rename Documents - LION_SHARED_DIR to LION_SHARED_DIR.

  • As is in LION_SHARED_DIR, make sure eGIS shared directory, namely, eGIS_PE, is available in your onedrive. To this end, visit eGIS_PE and click on Add shortcut to OneDrive to create a shortcut Documents - eGIS_PE in your OneDrive app directory. Optionally, rename Documents - eGIS_PE to eGIS_PE.

    NOTE: Double-click on Install_egis_app.bat batch file to install the eGIS application on your computer. The same batch file can be used to update future versions of the eGIS app as well

Once you are ready with eGIS setup, go to LION_SHARED_DIR directory and proceed with LION isntallation as follows:

  • Double-click on install_lion.bat and follow steps to install LION.

  • Press Next to continue

  • Press Install to continue

  • Please wait for the setup to complete

  • Setup is complete. If Launch LION checkbox is ticked, LION will be launched once Finish button is pressed.
